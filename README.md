# ❤️ Heart Disease Prediction using Hybrid NLP

🚀 Final Year Project using Machine Learning + Natural Language Processing (NLP)

---

## 📌 Project Overview
This project predicts heart disease risk by combining:
- Clinical patient data (age, BP, cholesterol, etc.)
- Natural language symptom analysis (NLP)

It uses a **hybrid approach**:
✔ Machine Learning model  
✔ NLP keyword-based scoring  

---

## 🎯 Features
- 🧠 ML-based prediction (Logistic Regression / Random Forest)
- 💬 NLP symptom analysis (TF-IDF)
- 📊 Risk classification (Low / Medium / High)
- 📈 Model performance visualization
- 🖥 Interactive Streamlit UI
- 📄 Report generation

---

## 🛠 Technologies Used
- Python
- Streamlit
- Pandas, NumPy
- Scikit-learn
- NLP (TF-IDF)
- Matplotlib
